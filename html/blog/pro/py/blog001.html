🔍 Pythonとは？

Python（パイソン）は、1991年にオランダ人プログラマーのGuido van Rossum (グイド・ヴァンロッサム)によって開発されたプログラミング言語です。名前の由来はBBCのコメディ番組『空飛ぶモンティ・パイソン』のパイソンから、名前を取りました。

Pythonの最大の特徴は「読みやすさ」と「書きやすさ」です。まるで英語の文章を読むように、コードの意味を理解することができます。

📝 Pythonのコードはこんな感じ

1. 基本的な挨拶プログラム

Python
# 名前を入力してもらう
    name = input("あなたの名前を教えてください: ")

    # 挨拶を表示
    print(f"こんにちは、{name}さん！")
    print("Pythonの世界へようこそ！")

2. リスト（配列）を使った処理

Python
# 果物のリストを作成
    fruits = ["りんご", "バナナ", "オレンジ", "ブドウ"]

    # すべての果物を表示
    for fruit in fruits:
        print(f"私は{fruit}が好きです！")

    # リストの長さを表示
    print(f"全部で{len(fruits)}種類の果物があります")

3. 関数を使った計算

Python
# 円の面積を計算する関数
    def calculate_circle_area(radius):
        pi = 3.14159
        area = pi * radius * radius
        return area

    # 関数を使って計算
    radius = 5
    area = calculate_circle_area(radius)
    print(f"半径{radius}cmの円の面積は{area}cm²です")

⚖️ 他のプログラミング言語と比較してみよう

同じ処理を他の言語と比較することで、Pythonの特徴がよく分かります。

「Hello, World!」を表示するプログラム

Python（シンプル！）
print("Hello, World!")
Java（少し複雑...）
public class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
C++（もっと複雑...）
#include <iostream>
    using namespace std;

    int main() {
        cout << "Hello, World!" << endl;
        return 0;
    }
Pythonなら1行で済むことが、他の言語では複数行必要になることがよくあります！
